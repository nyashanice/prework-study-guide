HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML creates webpage structure.
  • The style element defines CSS style for webpage.
  • The base element defines the base URL.
  • The link element connects external sources to webpage.
  • The meta element defines metadata like the character set, desctiption, keywords, author, etc.
  • All elements are case sensitive while declarations are not.
  • Attributes give additional information about an element and come at the beginning.
  • The !doctype declaration allows user to know what document to expect when opening.
  • h1-h6 are all header elements. The lower the number, the more prominent the text size.
  • The p element is paragraph form.
  • The ul element is an unordered list or bullet points.
  • The ol element is an ordered list or numbered list.
  • The li element is a list item so they're what goes in ul or ol elements.
  • Elements contain an opening and a closing tag.
  • Tags are only what's seen in brackets.
  • The img element contains information about an image.
  • The src attribute tells you where an image came from.
  • The alt attribute gives alternate text of an image for accesibility.
  • The br element creates a line break.
  • The a element is an anchor and provides a link to a website.
  • The href attribute directs you to the link for the anchor.

CSS

  • A margin indicated how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • How contents of a webpage are seen by users
  • Responsible for colors, aesthetics, layouts, organization
  • Inline CSS is a way to style element directly in HTML file
  • Inline CSS is quick and useful for permanent changes but less flexible than external and internal style sheets
  • Internal style sheets embed an entire stylesheet directly into HTML using a style element in the head element
  • External style sheet is most common
  • External CSS style sheet keeps all CSS rules in a separate file which makes design changes easier
  • With external CSS, file is linked in HTML head element
  • A selector defines element or attributes to which rules or declarations apply
  • Declarations are formatted as property:value;
  • * declares all elements on page to have the included style properties
  • Color can be assigned by using semantics (words) or hex codes

Git

  • git status: checks what branch we're currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • The checkout command moves a working branch to a new branch.
  • The -b flag creates a new branch.
  • git commands always begin with git
  • git add -A: adds changes in working branch to staging area
  • -A flag indicates we want to add all changes

JavaScript

  • Makes a webpage interactive
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A string is put inside quotation marks
  • Each line of JavaScript ends with a semicolon
  • The string (line of text) can be referred to by using variable name
  • Conditional statements can be used to interrupt control flow
  • Falsy values include 0, -0, On, "", null, undefined, NaN, or the Boolean value false
  • Truthy values include all other values and the Boolean value true
  • The strict equality operator "===" checks to see if two values are equal and returns a Boolean result of true if they are and false if they aren't
  • An if statement provides one or more conditions for the computer to evaluate
  • Arrays are useful when you're dealing with a large amount of data/values
  • An array is a single variable that is usd to hold a group of data
  • Arrays are usued to hold data that is related in some way
  • Arrays can hold multiple types of data: strings, numbers, etc.
  • An index is a number that identifies the location of a data item in an array
  • The first item in an array has an idex of 0
  • A for loop uses a pattern of indices to perform a task on all the items in an array
  • A for loop needs the shape's index and array's length
  • You can find array length by using built-in property of an array called length
  • For loop contains starting point for loop, condition, and statement that allows array to repeat over each item
  • Number in the variable for starting point has to be less than lenght of array for loop to run
  • x++ means x+1
  • Commenting out code makes it inactive without deleting it
  • To comment out multiple lines of code, use /* before and */ after
  • A function is a set of instructions that tells the computer how to perform a certain task
  • A function must be called by name to be executed, it doesn't run automatically with JS file
  • A function name is always followed by parentheses
